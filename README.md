# TCrypteur
Un logiciel de cryptage de texte par clé simple et facile d'utilisation.

# Présentation

TCrypteur est un petit outil de cryptage de texte en fonction d'une clé composé de chiffres.

Il a été codé avec python.
Liste des modules python utilisés :
      - Tkinter
      - time
      - random
      - unidecode

Lors du cryptage, le texte peut être légerement modifié :
      - suppression des accents et autres sur les lettres
      - les chiffres et caractères spéciaux sont laissés tel quel

J'ai créé ce projet sur mon temps libre, ce n'est donc pas super, mais tout de même pratique
si vous avez besoin de noter des choses que vous seul devez être capable de relire !

# Installation

Pour obtenir TCrypteur sur votre ordinateur, télechargez le dossier zip "TCrypteur_zip.zip",
puis décompressez-le.
